mser_extraction module¶

mser_extraction.extract_mser(img, args)¶

Function using mser module from the openCV library to extract maximally stable extremal regions. Additionally MSERs that lie inside others or have no neighbors are eliminated.

Parameters:
  • img (ndarray) – image from which MSERs are to be extracted
  • args (Arguments instance) –
  • filter (Bool) – whether to filter the extracted MSERs
Returns:

an instance of the Components class
