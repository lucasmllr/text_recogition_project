rot_correction module¶

rot_correction.correct_rot(img, args)¶

Function to rotate an input image according to the minimum entropy of its projection along the horizontal axis. Entropy is evaluated on a grid.

Parameters:
  • img (ndarray) – 2d image array
  • args (Arguments instance) – out of which the following are required args.n_angles (int): number of projection angles args.n_bins (int): number of bins for projection histograms. Should be odd. args.angle (float): abs of min and max angle (degrees) in whose boundaries n_angles projections are evaluated.
Returns:

An ndarray of the rotated image padded with zeros
