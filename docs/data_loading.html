data_loading module¶

class data_loading.ImageDataset(path='data', size=28)¶

Bases: torch.utils.data.dataset.Dataset

subclass of pytorch’s Dataset to provide extracted characters to a model.

path¶

String – path containing the data

raws¶

list of ndarrays – images stored in a list

truth¶

list of strings – ground truth strings of images

size¶

integer – height and width of input size for the model

blobs¶

list of ndarrays – contains size x size images of extracted letters

blob_truth¶

list of strings – ground truth for characters in blobs

extract_blobs()¶

function to perform blob extraction on raw images. Data is only used if blob extraction finds the right number of characters in an image.

load_raws()¶

loads raw images from the data directory and stores it as ndarrays

Returns:list of ndarrays
load_truth()¶

loads ground truth strings from “truth.txt” file

Returns:a list of strings
