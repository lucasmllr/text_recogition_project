component_evaluation.eliminate_tiny(components, threshold_factor=0.2)¶

eliminates components (in pace) that foll below a threshold of a minimum number of pixels determined from the median.

Parameters:

  • threshold_factor – facctor to multiply median by to get the threhsold
component_evaluation.filter_neighbors(components, args)¶

eliminates all components that have no neighbors as specified by the corresponding parameters in args. The components object is manipulated in place.

Parameters:
  • components – Components instance
  • args – Arguments instance
