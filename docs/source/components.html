class components.Component(region, bbox, img)¶

Bases: object

a class to hold a single component as found by the mser or blob extraction algorithm.

region¶

tuple of lists – as accepted by __init__

bbox¶

list – bounding box as accepted by __init__

x¶

float – upper left x coordinate of bbox (horizontal dimension)

y¶

flaot – upper left y coordinate of bbox (vertical dimension directed down)

w¶

float – width of bbox (horizontal)

h¶

float – height of bbox (vertical)

A¶

float – area of bbox

asp¶

float – aspect ration of bbox (w/h)

color¶

float – mean color, i.e. intensity of all pixels in region
