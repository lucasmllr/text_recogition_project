blob_extraction.find_blobs(raw_img, args)¶

function performing two dimensional connected component analysis on an image.

Parameters:
  • img (ndarray) – original image to be analyzed
  • args (Arguments instance) – defined the threshold value to binarze the image
Returns:

an instance of the Components class, a stencil containing the final labels of components, and a stencil containing the labels before eliminating equivalences
blob_extraction.get_bboxes(stencil, labels=None)¶

find the bounding boxes in a stencil of labeled regions. :param stencil: image with different pixel values for each region :type stencil: ndarray :param labels: region labels in the image

Returns:a list of bboxes in the format [x_min, x_max, y_min, y_max]
